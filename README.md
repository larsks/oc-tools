# Tools for making the OpenShift CLI a little easier

## Prerequisites

You will need to install [fzf](https://github.com/junegunn/fzf).

## Installation

Copy the `oc-*` scripts to a directory in your `$PATH`. E.g:

```
cp oc-* ~/bin/
```
